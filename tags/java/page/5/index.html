1. String常量池

1.1 创建String对象的两种方式

  • 通过new来创建String创对象，例如：String a = new String(“a”);

  • 直接将字符串常量赋值给一个对象引用，例如：String b = “b”;

    这两种不同的创建方法是有差别的，第一种方式是直接在Java heap内存空间创建一个新的对象，并且引用变量a指向这个对象，第二种方式是引用变量b指向常量池中的字符串。
